# Deployment Guide

## Prerequisites

- Node.js 16+
- MongoDB 4.4+
- Git

## Local Development Setup

1. Clone the repository
2. Install dependencies
3. Set up environment variables
4. Start MongoDB
5. Run the application

## Production Deployment

### Option 2: Heroku (Full Stack)
